# 沉浸场景图母提示词

帮我为这个产品制作沉浸场景图。
    沉浸式场景图追求**"无意识的心理账户归类"与"欲望的投影置换"**。它拒绝冷冰冰的参数陈列，而是构建一个能够引发镜像神经元共鸣的"平行时空"，其核心逻辑是让产品隐形于环境之中，却又成为美好生活的唯一钥匙。你应该思考如下三个核心点：
    1. 认知的软着陆（Cognitive Soft Landing）：在充满攻击性的广告噪音中，场景图是"舒适的避风港"。它不强迫大脑处理新奇刺激，而是利用大脑的"模式识别"机制，让用户在0.1秒内通过背景环境（如温馨的床头、高效的桌面）瞬间理解产品的定位，极大地降低了理解成本和防御心理。
    2. 价值的感官翻译（Sensory Translation of Value）：将抽象枯燥的技术参数（如"高织数"、"流明度"、"人体工学"）翻译成具象的感官体验。通过光影的质感暗示触觉，通过烟火气暗示嗅觉。它不是在说"这个产品很好"，而是在演示"拥有了这个产品，当下的体验有多美妙"。
    3. 预拥有的心理占有（Psychological Ownership）：场景图提供的是一种"试穿"体验。通过第一人称视角或极具生活气息的布局，诱导用户产生"这就是我的家"或"这就是我想要的生活"的错觉。这种错觉会产生一种失去感——如果不买，就仿佛失去了这种已经"预拥有"的美好生活。
    4. 注意防范图片里出现这些：
      a. 触发高压ul词汇：“power adapter”“adapter”“XXXv（高于36v都是高压）”“High Voltage”及电源插头图片
      b. 触发儿童cpc词汇：“kid”“child”“boy”“girl”及儿童图片
      c. 触发无线fcc词汇：“wireless”“bluetooth”“fcc”“phone control”及fcc标识、蓝牙标志，最需要注意的一点是有些产品上传了高压资质，然后插头上有fcc标识，然后触发了fcc无线资质
      d. 触发环保描述：“eco-friendly”“environmental friendly”“environment protection”
      e. 触发防尘防水描述：“outdoor””water proof“
      f. 不要涉嫌知识产权侵权（专利/商标/版权） 比如各类IP
      g. 严禁生成夜景/夜晚/傍晚/低照度场景（包括蓝色时刻、霓虹夜景、暗光室内）。所有画面必须是白天明亮环境，主体清晰通透，避免整体偏暗影响 Temu 转化。

## 规则

- 必须展示商品的全貌，不要细节特写。
- aspect ratio must be 100% 1:1
- 在营造场景氛围前，必须先应用使用机制与物理关系规则。场景中的商品必须处于真实可用的完成状态，而不是只被摆放在背景中。
- 对需要夹持、粘贴、悬挂、插入、卡扣、捆绑、穿戴或支撑的商品，必须清楚展示承载物、接触点、受力方向和正确完成状态。
- 不得出现悬浮、穿模、假接触、错误朝向、重心不稳、接口错位，或把夹具和挂钩误当作独立底座。
