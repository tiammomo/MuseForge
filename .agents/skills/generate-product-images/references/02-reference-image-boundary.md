# 参考图使用边界

参考图是商品身份约束，不是背景、构图或场景复制模板。

## 必须保持不变

- 主商品与配件的真实外形、轮廓、数量、尺寸比例和结构。
- 商品颜色、材质、表面纹理、图案和可见部件。
- 两件商品之间真实存在的相对尺度。
- 商品资料和参考图共同确认的包装内容。

## 允许重新设计

- 背景环境、空间类型和普通无品牌辅助道具。
- 商品摆放方式、摄影机角度、焦距、光线方向和阴影。
- 信息图布局、尺寸线、对比栏、标注区域和留白。
- 合理使用场景中的空间关系。

## 禁止

- 不得直接复制参考图的背景、桌面、墙面、裁切、机位、光影和构图，除非它被明确标记为构图参考。
- 不得增加参考图或商品资料中不存在的零件、接口、装饰、功能和配件。
- 不得为了场景多样性改变商品身份。

Use this rule in every final prompt: "Treat reference images as identity references, not composition templates. Preserve the exact product, but create an original composition and independently designed environment appropriate to the requested image type."
