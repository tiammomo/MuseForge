# 多样性规划规则

生成一批 prompt 前，先根据主商品资料选择场景域，再在兼容场景域内部做多样化。参考图只锁定商品身份，不能让原参考背景控制构图。

## 第一级：场景域

只能根据商品名称、类目、介绍、说明书、安装方式和真实用途选择：

- `home-living`：家居装饰、墙面用品、收纳和家庭生活用品。
- `kitchen-dining`：厨房、餐饮、备餐和餐具用品。
- `bath-personal-care`：洗漱、卫浴和个人护理用品。
- `office-study`：办公、学习、文件和桌面用品。
- `workshop-maintenance`：工具、五金、维修、维护和劳保用品。
- `vehicle-travel`：车辆、车载、行李、旅行和出行用品。
- `garden-daylight`：园艺、植物维护和庭院用品。
- `fitness-recreation`：运动、训练、健身和休闲用品。
- `pet-care`：宠物喂养、护理和生活用品。
- `retail-gifting`：礼品、纪念品或直接功能关系较弱的组合。
- `neutral-commercial`：无法从资料可靠判断用途时的默认场景。

无法可靠判断时必须使用 `neutral-commercial`，不得猜测用途。主商品决定主要场景域；配件只能影响组合动作和次要情境，不能把主商品带离兼容场景。

## 第二级：视觉变化

在已经选定的场景域内组合以下维度，相邻场景至少改变三个主要维度：

- 情境：日常使用、准备、安装完成、整理、清洁维护、收纳就绪、礼品组合或相邻使用。
- 视角：平视、轻俯拍、45度侧前方、接近第一人称、空间中景或适度顶前视角。
- 构图：中心锚定、三分法、对角线、对称框景、环境包围、前景引导、分层纵深或平衡非对称。
- 光线：左侧日光、右侧日光、顶部漫射日光、正面均匀日光、侧光加补光或高亮棚拍日光。
- 气质：整洁现代、温暖生活化、清爽高效、安静精致、实用可信或克制礼赠感。

## 外部日间环境

真实适合园艺、车辆、旅行、运动等商品时，可以使用明亮、干燥、清晰的日间外部环境。但图片文字和卖点不得出现敏感的环境适用或防水宣称，也不得通过雨、雪、积水、泥泞和恶劣天气暗示未经证明的性能。

禁止把敏感宣称限制错误理解成“所有商品必须放在室内”。同时严禁夜晚、傍晚、蓝色时刻、霓虹、暗光和低照度环境。

## 批次约束

1. 场景必须符合主商品真实用途，不允许从所有空间中随机选择。
2. 不允许仅通过更换背景颜色或小装饰物制造伪多样性。
3. 不得把工具放进卧室、把车辆用品放进梳妆区、把园艺用品放进办公桌等不兼容环境。
4. 直接功能关系弱时，选择中性商业、零售组合、收纳、相邻使用或礼赠场景，不虚构兼容性。
5. 场景变化不能改变商品颜色、数量、结构、材质和真实用途。
6. 同一商品和配件重新生成提示词时，场景分配必须稳定可复现。
